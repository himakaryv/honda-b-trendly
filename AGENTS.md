### Build, Lint, Test Commands
- Build: npm run build
- Lint: npm run lint
- Test: npm test
- Run single test: npm test -- <test-name>

### Code Style Guidelines
- Use consistent import order and grouping
- Format code with Prettier or an equivalent formatter
- Use explicit types for functions and variables
- Follow naming conventions: camelCase for variables and functions, PascalCase for types and classes
- Handle errors explicitly with try-catch blocks or error boundaries
